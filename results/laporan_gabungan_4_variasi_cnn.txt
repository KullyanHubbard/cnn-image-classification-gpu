1. Tujuan Eksperimen

Eksperimen ini dilakukan untuk membandingkan performa model CNN pada empat variasi kombinasi jumlah filter awal dan ukuran kernel. Tujuannya adalah melihat kombinasi mana yang menghasilkan akurasi terbaik pada data validasi dan data test.

Variasi yang dibandingkan:
1. Filter 32, kernel 3x3
2. Filter 32, kernel 5x5
3. Filter 64, kernel 3x3
4. Filter 64, kernel 5x5

2. Konfigurasi Umum

Seluruh variasi menggunakan konfigurasi umum yang sama agar hasilnya bisa dibandingkan secara adil.

- Ukuran gambar: 180 x 180
- Batch size: 64
- Validation split: 20%
- Epoch maksimum: 100
- Dataset training: 6400 gambar
- Dataset validasi: 1600 gambar
- Dataset test: 2000 gambar
- Jumlah gambar test kucing: 1000
- Jumlah gambar test anjing: 1000
- L2 regularization: 0.00001
- Optimizer: Adam
- Learning rate awal: 0.00025
- Loss function: binary_crossentropy
- Metric utama: accuracy
- Callback: EarlyStopping, ReduceLROnPlateau, dan ModelCheckpoint
- Augmentasi training: rescale, shear, zoom, rotation, width shift, height shift, dan horizontal flip
- Output layer: Dense 1 neuron dengan aktivasi sigmoid

5. Perbandingan dan Analisis

5.1 Berdasarkan Best Validation Accuracy

Urutan performa berdasarkan best validation accuracy:
1. Variasi 3, filter 64 kernel 3x3: 95.94%
2. Variasi 2, filter 32 kernel 5x5: 95.00%
3. Variasi 4, filter 64 kernel 5x5: 94.38%
4. Variasi 1, filter 32 kernel 3x3: 94.31%

Berdasarkan validasi, variasi 3 adalah yang paling baik. Variasi 2 juga cukup kuat, tetapi tidak mengalahkan variasi 3.

5.2 Berdasarkan Test Accuracy

Urutan performa berdasarkan test accuracy:
1. Variasi 3, filter 64 kernel 3x3: 95.40%
2. Variasi 1, filter 32 kernel 3x3: 94.85%
3. Variasi 4, filter 64 kernel 5x5: 94.70%
4. Variasi 2, filter 32 kernel 5x5: 94.50%

Berdasarkan data test, variasi 3 tetap menjadi model terbaik. Variasi 1 menjadi peringkat kedua pada test accuracy meskipun best validation accuracy-nya bukan yang tertinggi.

5.3 Berdasarkan Loss

Variasi 2 memiliki test loss paling rendah, yaitu 0.1711. Namun akurasi test variasi 2 hanya 94.50%. Ini berarti prediksi variasi 2 cenderung lebih yakin pada sebagian kasus, tetapi jumlah prediksi benar secara total masih kalah dari variasi 3.

5.4 Berdasarkan Keseimbangan Kelas

Variasi 3 memiliki performa per kelas paling seimbang:
- Recall kucing: 95.00%
- Recall anjing: 95.80%
- Selisih recall antar kelas: 0.80 poin

Variasi 1 dan variasi 4 lebih kuat mengenali anjing daripada kucing. Variasi 2 juga masih lebih kuat pada anjing, tetapi selisihnya lebih kecil daripada variasi 1 dan 4.

5.5 Berdasarkan Overfitting

Gap training-validation pada epoch validasi terbaik:
- Variasi 1: 4.89 poin
- Variasi 2: 3.34 poin
- Variasi 3: 3.42 poin
- Variasi 4: 5.20 poin

Variasi 2 memiliki gap paling kecil, tetapi test accuracy-nya tidak tertinggi. Variasi 3 tetap menjadi pilihan terbaik karena memiliki best validation accuracy dan test accuracy tertinggi, dengan gap yang masih cukup terkendali.

6. Kesimpulan Akhir

Berdasarkan empat variasi yang sudah diuji, model terbaik adalah Variasi 3 dengan filter 64 dan kernel 3x3. Model ini menghasilkan best validation accuracy 95.94% dan test accuracy 95.40%, sekaligus memiliki performa kucing dan anjing yang paling seimbang.

Hasil ini menunjukkan bahwa menambah jumlah filter dari 32 menjadi 64 dapat meningkatkan performa jika tetap menggunakan kernel 3x3. Namun, memperbesar kernel dari 3x3 menjadi 5x5 tidak selalu meningkatkan akurasi. Pada filter 64, kernel 5x5 justru menurunkan best validation accuracy dan test accuracy dibanding kernel 3x3.

Dengan demikian, konfigurasi yang paling direkomendasikan dari eksperimen ini adalah:
- Filter awal: 64
- Kernel size: 3x3
- Image size: 180 x 180
- Batch size: 64

Variasi 3 dapat dijadikan model terbaik dari eksperimen ini, sedangkan variasi lain dapat digunakan sebagai pembanding untuk menjelaskan pengaruh perubahan filter dan kernel terhadap performa CNN.
